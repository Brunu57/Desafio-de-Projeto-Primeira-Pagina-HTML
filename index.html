Inspetor de Elementos

(Início)

É um recurso encontrado em navegadores onde o usuário pode visualizar os códigos HTML e CSS que foram usados no para criar uma página da web. Conseguindo copiar esses conteúdos para fazer alterações em ambientes externos.

Estrutura Básica do HTML

(Início)

Nesta aula foi apresentada a criação de uma página HTML, primeiramente com o arquivo "index.html" onde serão colocadas as informações principais dela. Nesse arquivo somos introduzidos aos conceitos de "head" e "body", em "head" é colocado o titulo da página, ou seja, seu nome; em "body" são colocadas todas as informações que serão apresentadas ao usuário, como textos, imagens, listas e etc.

Falando Sobre Tags

(Início)

Uma tag é uma palavra-chave associada com uma informação que o descreve e permite uma classificação da informação baseada em palavras-chave. Nesta aula fomos apresentados à algumas tags:

  • < i >/< /i >: aqui os elementos entre as tags ficam em italico.
    •
  • < strong >/< /strong >: aqui os elementos entre as tags ficam em negrito.
    •
  • < input >: aqui será adicionado uma caixa, podendo ser de type="text", "color", "number".
    •
  • < img />: aqui será adicionado uma imagem à página, utilizando seu endereço.
    •

Atributos das Tags

(Início)

Atributos são informações passadas à Tag para que ela aja de maneira almejada. Existem atributos que funcionam em todas as Tags (globais) e os que são direcionados para cada Tag (específicos). Alguns deles são:

  • id=" ": este atributo serve como um identificador do elemento, para que ele possa ser encontrado em uma futura tag.
  • style=" ": este atributo serve para formatar um elemento, deixá-lo como o programador deseja (ex: cor).
  • class=" ": este atributo serve para padronizar os elementos, forma de usar algo em comum para facilitar o uso deles.
  • type=" ": este atributo é usado para alterar a propriedade da tag < input >, fazendo com que ela mude a caixa para: texto, númerica, de cor ou outras.
  • src=" ": este atributo serve para adicionar o endereço de uma imagem para a tag < img >, para que ela possa ser mostrada na página .
  • width=" ": este atributo serve para alterar o tamanho da imagem inserida na tag < img >.

Textos

(Início)

Aqui você encontrará tags utilizadas para criar e modificar textos, algumas são:

  • < h1, 2, 3, 4... >: utilizada para definir se o texto introduzido será um título, subtítulo ou menor.
    •
  • < p >: o texto inserido ficará no formato de parágrafo.
    •
  • < blockquote >: o texto inserido ficará em formato de uma citação, algo à parte do parágrafo.
    •
  • < u >: o texto inserido firará sublinhado.
    •
  • < mark >: o texto inserido firará destacado/marcado.
    •
  • < sup >: o texto inserido firará acima dos demais.
    •

Listas Ordenadas e Não Ordenadas

(Início)

Listas são utilizadas para organizar itens que possuem algo em comum, podendo ser ordanada ou não ordenada. Segue como criá-las via HTML:

  • < ol >/< /ol >: utilizada para criar uma lista ordenada, ou seja, onde os elementos são colocados em ordem específica.
    •
  • < ul >/< /ul >: utilizada para criar uma lista não ordenada, onde os elementos não seguem necessáriamentes uma ordem.
    •
  • < li >/< /li >: utilizada para criar um item dentro da lista, tanto ordenada quanto não ordenada, é aonde o texto será escrito.
    •

Links

(Início)

Aqui serão colocadas tag relacionadas ao uso de links nas páginas, tanto para extornos quanto para internos.

  • < a >Nome do Link< /a >: utilizada para criar a tag onde o link será colocado.
    •
  • href="": atributo utilizado para adicionar o link na tag anterior, podendo levar a sites externos, ou a páginas do mesmo usuário.
    •
  • title="":atributo utilizado para criar uma caixinha que dá nome/explica ou direciona o uso do link .
    •
  • target="": atributo utilizado para instruir o navegador a saber onde o link será aberto, podendo ser na própria página (_self) ou em uma nova guia (_blank).
    •
